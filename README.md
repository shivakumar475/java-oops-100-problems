# 🚀 100 Java OOP Practice Problems

This repository contains **100 Object-Oriented Programming (OOP)** practice problems written in Java.  
It is designed to strengthen your understanding of all core OOP concepts through practical, real-world examples.

Each question includes:

- ✔ A detailed problem statement  
- ✔ Well-structured Java solution  
- ✔ Clean OOP implementation  
- ✔ Class design with encapsulation  
- ✔ Real-world examples  

This project is perfect for **students, interview preparation, and Java beginners**.

---

## 📘 Topics Covered

This project covers the full OOP syllabus:

### 🔹 **1. Classes & Objects**
- Object creation  
- Constructors  
- Instance variables and methods  

### 🔹 **2. Encapsulation**
- Private variables  
- Getters & setters  
- Validation logic  

### 🔹 **3. Inheritance**
- Single inheritance  
- Multilevel inheritance  
- Hierarchical inheritance  

### 🔹 **4. Polymorphism**
- Method overloading  
- Method overriding  
- Runtime polymorphism  

### 🔹 **5. Abstraction**
- Abstract methods  
- Abstract classes  

### 🔹 **6. Interfaces**
- Interface implementation  
- Multiple interfaces  
- Interface inheritance  

### 🔹 **7. Composition & Aggregation**
- HAS-A relationship  
- Arrays/lists of objects  

### 🔹 **8. Real-World Problems**
- Bank accounts  
- Library systems  
- Student management  
- Employee records  
- Electricity bill  
- Insurance policy  
- Shopping cart  
- Hotel booking  
- Many more…

## 🛠️ How to Run the Programs

### **Option 1 — Using VS Code**
1. Open the project folder in VS Code  
2. Make sure Java extensions are installed  
3. Open any `.java` file  
4. Click **Run ▶** in the top-right corner
